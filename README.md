# Flashcards App

Aplicación simple de tarjetas educativas de estudio. Selecciona un tema, haz clic en una tarjeta para ver pregunta/respuesta.

## Estructura del Proyecto

- `public/index.html`: Página principal.
- `src/css/style.css`: Estilos.
- `src/js/data/flashcards.js`: Datos de ejemplo (temas y tarjetas).
- `src/js/app.js`: Lógica principal.

## Uso

1. Abre `public/index.html` en tu navegador.
2. Haz clic sobre los temas y las tarjetas para estudiar.

¡Ideal para ampliar con nuevos temas y funcionalidades!
